# Personal Website

A minimal personal website in the style of Paul Graham and Nat Friedman, built with React.

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Features

- Simple, minimal design inspired by early 2000s web aesthetics
- Clean typography with Georgia serif font
- Text-focused layout
- No JavaScript frameworks beyond React
- Classic web design principles
